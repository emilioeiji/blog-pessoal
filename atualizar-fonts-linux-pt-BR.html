Atualizando Fontes no Linux: Mantendo seu Texto com Visual Atraente

Written on dez 09, 2018

Quando se trata de design e legibilidade, a escolha das fontes é crucial. No entanto, depois de instalar novas fontes no seu sistema Linux, você também precisa se certificar de que elas sejam atualizadas e disponíveis para todas as suas aplicações. Neste guia, vamos explorar como atualizar as fontes após instalá-las para garantir um visual atraente e consistente em todo o sistema.

Atualizando o Cache de Fontes:

Após instalar novas fontes, é importante atualizar o cache de fontes do sistema para que as aplicações possam reconhecê-las corretamente. Siga os passos abaixo para fazer isso:

  1. Abra o terminal.

  2. Caso não existir o diretório .fonts na raiz do seu horme, deve-se criá-lo:

    mkdir ~/.fonts

  3. Coloque os novos arquivos dentro do diretório .fonts.

  4. Use o seguinte comando para atualizar o cache de fontes:
    fc-cache -fv

Esse comando regenera o cache de fontes com base nas fontes instaladas no sistema.

Verificando a Atualização:

Para verificar se as fontes foram atualizadas corretamente, você pode usar o seguinte comando:

fc-list

Isso exibirá uma lista de todas as fontes disponíveis no sistema, incluindo as que você acabou de instalar.

Aplicando as Fontes Atualizadas:

Depois de atualizar o cache de fontes, as novas fontes devem estar disponíveis em suas aplicações. Você pode selecioná-las e aplicá-las de acordo com suas preferências:

  1. Vá para as configurações de fonte da sua aplicação, seja no sistema ou nas configurações específicas da aplicação.
  2. Procure a lista de fontes disponíveis e selecione a fonte atualizada que você deseja usar.
  3. Aplique a fonte e observe a mudança no visual do texto.
  4. No meu caso em que estava utilizando o GIMP, tive fechar e abrir a aplicação novamente para surtir efeito.
